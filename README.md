# 烟花产品信息展示 H5 页面 (Fireworks Info H5)

这是一个适配移动端（iPhone 13-17 兼容）的烟花产品展示 H5 页面。具备分类导航、产品详述、大图预览及视频演示功能。

## 🌟 核心特性

- **移动优先**：深度适配主流手机屏幕，支持视口安全区域（Safe Area）。
- **高端视觉**：采用现代深色主题，金/红配色风格，衬托烟花产品美感。
- **动态交互**：
    - 左侧分类导航，支持“全部”展示。
    - 产品图片点击放大预览（带关闭按钮）。
    - 智能标签展示（网红/大牌/推荐等）。
    - 视频播放快捷跳转。
- **性能优异**：基于 `data.js` 静态数据加载，无需后端服务，部署即用。

## 📊 数据更新说明

本项目的数据源为 `本地`。若需更新页面内容：

1. 编辑并保存 `本地数据`。
2. 运行数据转换脚本：
   ```bash
   python3 convert_data.py
   ```
3. 脚本会自动重新生成 `data.js` 及 `data.json`。
4. 刷新网页或重新提交代码即可生效。

## 📂 目录结构

- `index.html`: 网页主入口。
- `style.css`: 页面样式逻辑。
- `app.js`: 页面交互驱动。
- `data.js`: 当前展示的数据库脚本（由 Excel 导出）。
- `files_img/`: 存放所有产品封面的图片库。
- `convert_data.py`: 处理 Excel 到 JS 的转换工具。
